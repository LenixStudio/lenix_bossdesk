# lenix_bossdesk
AKA management, for managers with control over finance, members and more

## About
### Introduction
Welcome to Boss Desk

This script is designed to enhance your server's gameplay by:
* Creating a boss desk that allows players that are managers to manage their inventory, finance, members, and other features

### Instructions
* Installation: drag and drop
* Environment: QBox
* Support: available
